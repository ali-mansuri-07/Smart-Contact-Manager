Welcome to Smart Contact Manager Application

The Smart Contact Manager is a responsive web application developed using HTML, CSS, JavaScript, Thymeleaf, Spring Boot, and MySQL. It provides users with authentication and authorization features using Spring Security. Users can sign up and log in to manage their contacts securely.

Key features include:

  • User authentication and authorization
  • Create, update, view, and delete contacts
  • Change password using OTP through email
